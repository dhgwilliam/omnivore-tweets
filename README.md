# getting started

  * `git clone https://github.com/dhgwilliam/omnivore-tweets.git`
  * `cd omnivore-tweets`
  * `bundle install`
  * (OPTIONAL) add your bit.ly login and [api key](https://bitly.com/a/settings/advanced) to `.env`:

        API_LOGIN='dhgwilliam'
        API_KEY='R_abcdef0123456789'

  * `bin/omnivore -h`
  * `bin/omnivore update`
